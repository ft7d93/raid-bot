# Discord raid bot 

Paremeters to change:

Go to the file `\config.json` and change the parameters: 

```json
{
    "raidername": "RAIDERNAME",
    "token": "TOKEN", 
    "guildicon": "ANYIMGURL",
    "prefix": "!!!"
}

```
# For start the bot

Open `\start.bat`
